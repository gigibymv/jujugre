# Jujugre + minimalist-ui

When both skills apply:

- **Icons:** Keep `lucide-react` for navigation density and consistency until a deliberate icon pass.
- **Palette:** The warm cream / sage / dusty rose system is product identity — refine tokens, do not switch to cold gray SaaS defaults.
- **Gradients:** Remove decorative gradients on chrome (nav mark, badges); keep flat fills and 1px borders.
- **Emojis:** Banned in UI strings; use Lucide or plain text labels.
